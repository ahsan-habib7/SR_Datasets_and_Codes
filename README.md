# Defect Detection and Classification Dataset (Defect.csv)

## Overview
`Defect.csv` is a consolidated dataset designed for training and evaluating machine learning, deep learning, and Natural Language Processing (NLP) models in the domain of **Software Requirements Engineering**. Specifically, it targets the tasks of **Defect Detection** and **Classification** within software requirement specifications.

The dataset consists of a single foundational column: `Requirement Text`. It serves as a unified benchmark by merging, cleaning, and aggregating data from five distinct, publicly available requirements engineering datasets.

---

## Dataset Provenance & Sources
This dataset is a curated compilation of the following five public repositories:

1. **GitReq Artifacts**  
   * **Source/DOI:** [https://doi.org/10.6084/m9.figshare.31669477.v3](https://doi.org/10.6084/m9.figshare.31669477.v3)
   * **Description:** Contains requirement artifacts, issues, and descriptions harvested from GitHub repositories, capturing real-world developer syntax and formatting.

2. **FR_NFR_dataset**  
   * **Source/DOI:** [doi: 10.17632/4ysx9fyzv4.1](https://doi.org/10.17632/4ysx9fyzv4.1)
   * **Description:** A classic repository separating Functional Requirements (FR) and Non-Functional Requirements (NFR), heavily utilized for requirement classification benchmarks.

3. **Pure_Annotate_Dataset**  
   * **Source:** Kaggle
   * **Description:** A dataset originating from the PURE (Public Requirements Dataset) corpus, annotated for ambiguity, language defects, or specific architectural concerns.

4. **PROMISE-relabeled-NICE**  
   * **Source/DOI:** [doi: 10.5281/zenodo.14590935](https://doi.org/10.5281/zenodo.14590935)
   * **Description:** An updated, rigorously cleaned, and relabeled variant of the foundational PROMISE requirements repository, refined to match modern software engineering ontology and quality metrics.

5. **software-requirements-dataset**  
   * **Source:** Kaggle
   * **Description:** A diverse collection of software requirement specifications across various industrial domains, representing different styles of structured and unstructured text.
